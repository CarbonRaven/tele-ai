# Raspberry Pi 5 Technical Documentation

## Overview

The Raspberry Pi 5 is the latest flagship single-board computer from the Raspberry Pi Foundation, released in October 2023. It represents a significant upgrade over the Raspberry Pi 4, featuring custom silicon developed in-house and substantial performance improvements.

## Key Highlights

- **First Pi with custom silicon**: The RP1 "southbridge" chip provides I/O capabilities, delivering improved peripheral performance
- **2-3x CPU performance improvement** over Raspberry Pi 4
- **New features**: PCIe 2.0 interface, power button, real-time clock (RTC)
- **Enhanced display support**: Dual 4Kp60 HDMI with HDR

## Physical Dimensions

- Standard Raspberry Pi form factor
- 40-pin GPIO header (same as Pi 4)
- Built-in fan header and mounting points for active cooling

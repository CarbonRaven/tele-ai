# Iconic Telephone Services of the 1980s and 1990s

A comprehensive reference of telephone services, features, and cultural touchstones from the golden age of landline telephony. These services can inspire features for the AI payphone project.

---

## Directory & Information Services

### 411 - Directory Assistance
- **How it worked**: Dial 411, speak to a live operator, give them a name and city
- **Cost**: Initially free, later $0.50-$1.50 per call
- **Experience**: "What city and state? What listing?" followed by automated number readback
- **Fun fact**: Operators could look up by address or business type, not just name

### 555-1212 - Long Distance Directory
- **Format**: 1-[area code]-555-1212
- **Purpose**: Find phone numbers in other area codes
- **Cost**: Typically $0.75-$1.25 per call

### Time & Temperature Services
- **Common number**: POPCORN (767-2676) in many cities
- **Sponsors**: Local banks, jewelers ("At the tone, the time will be 3:45 PM, brought to you by First National Bank")
- **Format**: "At the tone, the time will be [time] and [seconds] seconds. The temperature is [temp] degrees."
- **Cultural note**: Before smartphones, this was how you set your watch

### Weather Services
- **Local numbers**: Varied by city
- **Content**: Current conditions, forecast, sometimes pollen count
- **Sponsors**: TV stations, radio stations, local businesses
- **Extended forecasts**: Marine weather, ski conditions in relevant areas

## Movie & Ticket Services

### Moviefone (777-FILM)
- **Launched**: 1989
- **Famous greeting**: "Hello, and welcome to Moviefone!"
- **Voice**: Russ Leatherman (Mr. Moviefone)
- **How it worked**:
  1. Enter ZIP code
  2. Press numbers for theater
  3. Press numbers for movie
  4. Hear showtimes
- **Cultural impact**: Parodied in Seinfeld ("Why don't you just tell me the name of the movie you want to see?")
- **Peak usage**: 3 million calls/week in the 90s

### Ticketmaster Phone Sales
- **Service**: Buy concert and event tickets by phone
- **Experience**: Long hold times, "convenience fees"
- **Competition**: Ticketron (until 1991)

### Local Theater Hotlines
- **Format**: Recorded messages listing current films and showtimes
- **Update frequency**: Usually weekly

## Collect Call & Long Distance Services

### Traditional Collect Calls
- **How to place**: Dial 0 + number, tell operator your name
- **Famous phrase**: "You have a collect call from [name]. Will you accept the charges?"
- **The hack**: Say your message as your "name" to avoid charges ("Bob-I'm-at-the-airport-pick-me-up")

### 1-800-COLLECT (1993)
- **Marketing**: Heavy celebrity endorsements (David Arquette, Mr. T)
- **Slogan**: "Save a buck or two"
- **Competition**: Direct challenge to AT&T collect

### 1-800-CALL-ATT
- **AT&T's response**: To 1-800-COLLECT
- **Celebrity endorsers**: Various

### 10-10 Dial-Around Numbers
| Code | Company | Slogan/Celebrity |
|------|---------|------------------|
| **10-10-220** | Telecom USA | John Lithgow |
| **10-10-321** | MCI | Various |
| **10-10-811** | Lucky Dog | "Dial and save" |
| **10-10-345** | Excel | Terry Bradshaw |
| **10-10-987** | Various | Hulk Hogan, ALF |

**How they worked**: Dial the code before the number to use a different (cheaper) long distance carrier.

### Calling Cards
- **Purchase locations**: Gas stations, convenience stores, phone kiosks
- **Denominations**: $5, $10, $20, $50
- **How to use**:
  1. Dial 1-800 access number
  2. Enter PIN from scratch-off
  3. Dial destination number
- **Brands**: AT&T, MCI, Sprint, prepaid cards

## Telephone Features & Star Codes

### Call Management Features

| Feature | Code | Description |
|---------|------|-------------|
| **Call Return** | *69 | Dial back last incoming caller |
| **Call Block** | *60 | Block specific numbers from calling you |
| **Caller ID Block** | *67 | Hide your number on next outgoing call |
| **Per-Line Blocking** | *82 | Unblock your number for next call |
| **Selective Call Forwarding** | *63 | Forward calls from specific numbers |
| **Call Forwarding** | *72 | Forward all calls to another number |
| **Cancel Call Forwarding** | *73 | Turn off call forwarding |
| **Three-Way Calling** | Hook flash + dial | Add a third party to call |
| **Call Waiting Disable** | *70 | Turn off call waiting for next call |

### Call Waiting
- **Introduced**: 1970s, widespread by 1980s
- **Sound**: Distinctive "beep" during call
- **Operation**: Hook flash to switch between calls
- **Social impact**: "Can you hold on? I have another call"

### Caller ID
- **Widespread adoption**: Early 1990s
- **Display**: Small box or built into phone
- **Information**: Number, later name
- **Cultural shift**: End of anonymous calling

### Voicemail
- **Replaced**: Answering machines (late 80s-90s)
- **Carrier-based**: Messages stored at phone company
- **Access**: Dial your own number or special code
- **Features**: Multiple mailboxes, greetings, forwarding

## Answering Machines (Pre-Voicemail)

### The Experience
- **Outgoing message**: "Hi, you've reached the Smiths. We can't come to the phone right now..."
- **The beep**: "Please leave a message after the tone. BEEP."
- **Screening calls**: Listening while someone left a message, deciding whether to pick up
- **Tape limitations**: Finite message space, rewinding to listen

### Cultural Moments
- **Clever outgoing messages**: Singing, jokes, celebrity impressions
- **"The machine ate the tape"**: Excuse for not returning calls
- **Blinking light**: Anticipation of who called

## Operator Services

### Live Operators
| Service | How to Reach | Purpose |
|---------|--------------|---------|
| **Local Operator** | 0 | General assistance, emergencies |
| **Long Distance Operator** | 00 | Placing LD calls, person-to-person |
| **International Operator** | 01 | International call assistance |
| **Directory Assistance** | 411 | Phone number lookup |

### Person-to-Person Calls
- **Concept**: Only pay if specific person answers
- **Use case**: Long distance to shared phone (dorms, offices)
- **How it worked**: Operator asks for specific person by name

### Emergency Services
- **911**: Established 1968, widespread by 1980s
- **Before 911**: Dial 0 for operator, ask for police/fire/ambulance

## Pager Services (Adjacent Technology)

### How Pagers Worked
1. Someone calls your pager number
2. They enter a callback number or short code
3. Pager beeps/vibrates and displays the number
4. You find a phone and call back

### Pager Codes
| Code | Meaning |
|------|---------|
| **143** | I love you (1 letter, 4 letters, 3 letters) |
| **411** | Info / What's up? |
| **911** | Emergency / Call ASAP |
| **07734** | "HELLO" upside down |
| **1134** | "HELL" upside down |
| **823** | Thinking of you |
| **637** | Always and forever |

### Pager Culture
- **Demographics**: Doctors, drug dealers, teenagers
- **Fashion**: Worn on belt, lanyard
- **Two-way pagers**: Could send text replies (late 90s)

### Telephone Sounds

| Sound | Description |
|-------|-------------|
| **Dial tone** | Steady hum indicating line ready |
| **Busy signal** | Rhythmic beeping (engaged) |
| **Ring-back tone** | What you hear while their phone rings |
| **Off-hook tone** | Loud, annoying tone when phone left off hook |
| **Reorder tone** | Fast busy signal (network problem) |
| **Touch-tone beeps** | DTMF tones when dialing |
| **Rotary dial clicks** | Mechanical pulse dialing |
| **Coin drop** | Quarter falling into payphone |
| **Coin return** | Rejected coin falling to return |

### Modem Sounds (Dial-Up Era)
The iconic handshake sequence:
1. Dial tone
2. Touch-tone dialing
3. Ring
4. Answer tone (carrier)
5. Negotiation screeches and chirps
6. Silence (connected)
